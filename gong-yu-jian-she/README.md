# 公寓建设

## **公寓房间必填属性有**

1. 房间名
2. 位置：区域、楼栋、楼层
3. 维护人

即在创建房间之前，必须要有位置和维护人

之前已经创建过了管理账号

那么公寓建设就需先建设房间位置
